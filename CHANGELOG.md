# Change Log

## [1.1.0]
- Overhauled the statistics viewer into a clean, terminal-style dashboard with ASCII bar charts
- Added milestone notifications to celebrate your saving achievements
- Improved performance by caching saves in memory and throttling disk writes
- Fixed tracking of virtual files (e.g., output panels, Git diffs)
- Fixed the daily average calculation to accurately reflect total days since installation
- Fixed the activation event so the extension correctly tracks all files on startup
- Made the statistics preview document read-only
